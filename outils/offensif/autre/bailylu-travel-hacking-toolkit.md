## Description
Cet outil est conçu pour collecter des informations ouvertes (OSINT) dans le cadre d'attaques ciblant les systèmes de réservation ou les comptes de voyage. Il peut automatiser l'extraction de données sensibles liées aux itinéraires, aux profils utilisateurs ou aux programmes de fidélité.
