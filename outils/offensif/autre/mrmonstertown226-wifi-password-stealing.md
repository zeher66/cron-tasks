## Description
Cet outil permet de récupérer les identifiants Wi-Fi sauvegardés sur un système cible, facilitant l'accès non autorisé aux réseaux sans fil. Il s'inscrit dans les activités d'audit de sécurité axées sur l'extraction de mots de passe.

A network security auditing tool. It allows for the recovery of saved Wi-Fi credentials and helps analyze system configuration vulnerabilities effectively.
