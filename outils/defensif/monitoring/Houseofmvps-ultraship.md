## Description
Ultraship est un plugin pour Claude Code intégrant 29 outils d'audit automatisés couvrant la sécurité, la qualité du code et les performances avant déploiement. Il permet de surveiller et renforcer la qualité et la sécurité du logiciel en amont de la production.

Ultraship — Claude Code plugin. 32 expert-level skills for building, shipping, and scaling production software. 29 audit tools (security, code quality, bundle size, SEO/GEO/AEO) close the loop before deploy.
